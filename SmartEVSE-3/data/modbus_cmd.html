Modbus Command

Send a Modbus command to a device on the RS485 bus. The command is queued immediately and executed as soon as the bus is free.

Read Registers — FC=3 / FC=4

Example: Acrel ADL400 sub-meter — read voltage/current block: address 12, register 97, count 10, function 4 (or 3 — the ADL400 only implements 03H, so use function 3)

Write Single Register — FC=6

Example: Lixee TIC DIN — activate WiFi: address 1, register 666, value 1

Write Multiple Registers — FC=16

Enter up to 8 register values (0–65535), one per line, starting at the given register address.
